Node.js é um ambiente de execução Javascript de codigo aberto e multiplataforma, construído sobre o motor V8 do Chrome. Ele permite que você execute Javascript no lado do servidor, ou seja, fora do navageador, possibilitando o desenvolvimento aplicações WEB e outras ferramentas Back-End. Node.js é um software de codigo aberto, multiplataforma, baseado no interpretador V8 do Google e que permite a execução de codigos Javascript fora de um navegador WEB. A principla característica do Node.js é sua arquitetura assíncrona e orientada por eventos.
